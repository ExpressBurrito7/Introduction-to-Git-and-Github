Q 1) In order to get the contents of a remote branch without automatically merging, which of these commands should we use?
Resp) git remote update

Q 2) If we need to find more information about a remote branch, which command will help us?
Resp) git remote show origin

Q 3) What command will download remote branches from remote repositories without merging the content with your current workspace automatically?
Resp) git fetch

Q 4) What type of merge creates a new merge commit?
Resp) Explicit merge

Q 5) What method of getting remote contents will automatically merge the remote branch with the current local branch?
Resp) git pull
