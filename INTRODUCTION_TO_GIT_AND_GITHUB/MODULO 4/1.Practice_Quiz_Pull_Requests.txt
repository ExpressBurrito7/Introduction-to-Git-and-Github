Q 1) What is the difference between using squash and fixup when rebasing?
Resp) Squash combines the commit messages into one. Fixup discards the new commit message

Q 2) What is a pull request?
Resp) A request send to owner and collaborators of the target repository to pull your recent changes

Q 3) Under what circumstances is a new fork created?
Resp) When you want to experiment with changes without affecting the main repository .
